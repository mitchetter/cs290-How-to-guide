How-To Guide: Using the OMDb API

Mitch Etter, CS290-400, Spring 2015


Useage


This guide will illustrate how to make the basic API calls in PHP. One thing to note is that you do not need an API key to use the data only service call. According to the publisher, this very well may change in the near future. If you want to make poster API requests, you do need an API key. To gain an API key for the poster content, you will need to access this page and request a key. Please note that the key will not become active until you donate to the website. All site content is contributed, maintained and tested by the service users. Donators are considered testers to the poster API and therefore are the only users who have valid access to the API. I donated only $10 USD and I considered it a small price to pay to be able to play with this feature. You can get the poster URL free without a key but not direct access to the image without the key. The form for API key access looks like this:



Here is the syntax of the two calls that can be made:

Data Only API Request

            http://www.omdbapi.com/?

Poster API Request

            http://img.omdbapi.com/?apikey=[yourkey]&
